# Intégrations Zoho - Guide Complet

## Introduction

Ce guide couvre les méthodes d'intégration transversales de l'écosystème Zoho avec des systèmes externes : API REST, webhooks, OAuth 2.0 et connexions avec des services tiers.

```
┌─────────────────────────────────────────────┐
│           Méthodes d'intégration            │
├────────────┬────────────┬──────────────────┤
│  API REST  │  Webhooks  │  Zoho Flow       │
│ (pull)     │ (push)     │  (no-code)       │
├────────────┼────────────┼──────────────────┤
│  OAuth 2.0 │  Deluge    │  Connexions      │
│ (auth)     │ (scripts)  │  tierces         │
└────────────┴────────────┴──────────────────┘
```
