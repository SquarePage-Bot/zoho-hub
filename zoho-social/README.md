# Zoho Social - Guide Complet

## Introduction

Zoho Social est un outil de gestion des réseaux sociaux qui permet de planifier, publier, surveiller et analyser votre présence sur les principales plateformes sociales depuis une interface unique.

## Plateformes Supportées

| Plateforme | Publication | Monitoring | Analytics |
|-----------|-------------|-----------|-----------|
| Facebook (Pages) | ✅ | ✅ | ✅ |
| Instagram (Business) | ✅ | ✅ | ✅ |
| Twitter / X | ✅ | ✅ | ✅ |
| LinkedIn (Pages & Profils) | ✅ | ✅ | ✅ |
| Google Business Profile | ✅ | ❌ | ✅ |
| Pinterest | ✅ | ❌ | ✅ |
| TikTok | ✅ | ❌ | ✅ |
| YouTube | ✅ | ❌ | ✅ |

## Fonctionnalités Clés

- **Publication** : Planification, file d'attente, recyclage de contenu
- **Monitoring** : Surveillance des mentions, mots-clés, concurrents
- **Engagement** : Répondre aux commentaires et messages depuis un seul endroit
- **Analytics** : Rapports de performance, meilleurs horaires
- **Collaboration** : Rôles d'équipe, approbations, commentaires internes
- **Intégration CRM** : Lier les interactions sociales aux contacts Zoho CRM
