# Zoho Social - Rapports et Analyses

## Rapports par Plateforme

### Métriques Communes
| Métrique | Description |
|---------|-------------|
| Portée (Reach) | Nombre de personnes ayant vu le contenu |
| Impressions | Nombre total d'affichages |
| Engagement | Likes + commentaires + partages + clics |
| Taux d'engagement | Engagement / Portée × 100 |
| Clics sur le lien | Nombre de clics vers votre site |
| Croissance abonnés | Nouveaux abonnés sur la période |

### Exemple de Rapport Mensuel
```
=== Rapport Février 2026 ===

  Engagement : 3 240 (taux : 7.2%)
  Meilleur post : "Annonce V2.0" (portée : 12 500)
```

## Rapports Personnalisés

### Créer un Rapport
```
1. Rapports → Nouveau rapport
2. Sélectionner les plateformes
3. Choisir la période
4. Sélectionner les métriques
5. Ajouter des comparaisons (période précédente, concurrent)
6. Planifier l'envoi automatique (PDF par email)
```

### Rapport de Comparaison Concurrentielle
```
Votre marque vs Concurrent A vs Concurrent B

| Métrique | Vous | Concurrent A | Concurrent B |
|---------|------|-------------|-------------|
| Abonnés FB | 12 500 | 18 200 | 9 800 |
| Engagement FB | 7.2% | 5.1% | 8.5% |
| Posts/semaine | 5 | 7 | 3 |
| Temps de réponse | 45 min | 2h | 30 min |
```

## Rapports Automatisés

```
Planification :
  Rapport hebdomadaire : Chaque lundi à 9h → Email au manager
  Rapport mensuel : Le 1er du mois → Email à l'équipe marketing
  Rapport trimestriel : PDF détaillé → Direction

Format : PDF, CSV, ou lien en ligne
```

## Bonnes Pratiques

1. **Suivre les KPIs pertinents** — pas toutes les métriques, les plus utiles
2. **Comparer période vs période** pour identifier les tendances
3. **Analyser les meilleurs posts** et reproduire les formats gagnants
4. **Automatiser les rapports** pour gagner du temps
5. **Corréler avec les objectifs business** (trafic site, leads, ventes)
