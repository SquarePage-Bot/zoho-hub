# Zoho Sign - Guide Complet

## Introduction

Zoho Sign est une solution de signature électronique légalement reconnue qui permet de signer, envoyer et gérer des documents numériques. Conforme aux réglementations eIDAS (Europe), ESIGN Act (USA) et IT Act (Inde).

## Fonctionnalités

- **Signature électronique** : Signature simple, avancée et qualifiée
- **Templates** : Modèles réutilisables pour les documents récurrents
- **Workflows** : Ordre de signature, approbations, rappels
- **Piste d'audit** : Traçabilité complète de chaque action
- **Intégrations** : Zoho CRM, Zoho Books, Google Drive, etc.
- **API** : Automatisation des envois et signatures

## Types de Signatures

| Type | Niveau | Valeur juridique | Exemple |
|------|--------|------------------|---------|
| Simple | Basique | Standard | Clic "J'accepte" |
| Avancée | Intermédiaire | Renforcée | Signature dessinée + vérification email |
| Qualifiée | Élevé | Équivalent manuscrit | Certificat qualifié eIDAS |
