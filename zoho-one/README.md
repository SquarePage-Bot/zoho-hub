# Zoho One - Guide Complet

## Introduction

Zoho One est la suite tout-en-un de Zoho qui regroupe plus de 45 applications intégrées pour gérer l'ensemble d'une entreprise : CRM, finance, RH, marketing, support, projets, analytics et plus. Un seul abonnement, un seul panneau d'administration.

## Applications Incluses (Sélection)

| Catégorie | Applications |
|-----------|-------------|
| Ventes & CRM | CRM, SalesIQ, Bookings, Forms |
| Marketing | Campaigns, Social, Survey, PageSense |
| Finance | Books, Invoice, Inventory, Expense, Subscriptions |
| Support | Desk, Assist, Lens |
| RH | People, Recruit, Workerly |
| Projets | Projects, Sprints, BugTracker |
| Communication | Mail, Cliq, Meeting, Connect |
| Productivité | Writer, Sheet, Show, WorkDrive |
| Analytics | Analytics, DataPrep |
| Développement | Creator, Flow, Catalyst |
| Sécurité | Vault, Directory, OneAuth |
