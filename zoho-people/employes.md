## Onboarding (Intégration)

### Checklist d'Onboarding
```
Avant le jour J :
  ☑ Créer le compte email
  ☑ Préparer le poste de travail
  ☑ Commander le badge d'accès
  ☑ Envoyer le kit de bienvenue
  ☑ Préparer les documents à signer

Jour 1 :
  ☑ Accueil par le manager
  ☑ Tour des locaux
  ☑ Signature du contrat
  ☑ Remise du matériel
  ☑ Configuration des accès IT

Semaine 1 :
  ☑ Formation outils internes
  ☑ Rencontre avec l'équipe
  ☑ Présentation des projets en cours
  ☑ Assignation du mentor/buddy

Mois 1 :
  ☑ Point avec le manager
  ☑ Feedback d'intégration
  ☑ Objectifs de la période d'essai
```

## Offboarding (Sortie)

### Processus de Départ
```
1. Notification de départ (démission/licenciement)
2. Entretien de départ
3. Transfert des responsabilités
4. Récupération du matériel
5. Désactivation des accès
6. Solde de tout compte
7. Remise des documents (certificat, attestation Pôle Emploi)
8. Archivage du dossier
```

## Champs Personnalisés

Zoho People permet d'ajouter des champs spécifiques :

```
Exemples de champs personnalisés :
- Taille de t-shirt (pour les goodies)
- Permis de conduire (Oui/Non)
- Langues parlées (multi-sélection)
- Étage / bureau
- Place de parking
- Allergies alimentaires (pour les événements)
```

## Gestion Documentaire

### Documents par Employé
```
Types de documents :
- Contrat de travail (signé)
- Avenants
- Pièce d'identité
- RIB
- Justificatif de domicile
- Diplômes
- Certificats de formation
- Avis d'arrêt maladie
- Bulletins de paie

Stockage :
- Dossier par employé
- Accès restreint (RH + employé concerné)
- Durée de conservation configurable
```

## Bonnes Pratiques

1. **Compléter les fiches à 100%** : chaque champ renseigné facilite les analyses
2. **Mettre à jour en temps réel** les changements (poste, salaire, manager)
3. **Automatiser l'onboarding** avec des workflows
4. **Restreindre les accès** : chaque employé ne voit que ses données
5. **Archiver les dossiers** des employés sortis (conformité RGPD)
