# Zoho People - Évaluations de Performance

## Types d'Évaluations

### 1. Évaluation Annuelle
```
Fréquence : 1 fois/an (généralement en décembre/janvier)
Participants : Employé + Manager
Durée du cycle : Année civile ou fiscale
Contenu :
  - Bilan des objectifs
  - Compétences comportementales
  - Plan de développement
  - Discussion sur la rémunération
```

### 2. Revue Semestrielle / Trimestrielle
```
Fréquence : 2-4 fois/an
Plus légère que l'évaluation annuelle
Focus : avancement des objectifs, ajustements
```

### 3. Feedback Continu
```
À tout moment, entre pairs ou manager-employé
Types :
  - Feedback positif (reconnaissance)
  - Feedback constructif (amélioration)
  - Feedback projet (fin de mission)
```

### 4. Évaluation 360°
```
Sources de feedback :
  - Auto-évaluation
  - Manager direct
  - Pairs (collègues)
  - Subordonnés
  - Clients internes/externes (optionnel)

Résultat : Vue complète des compétences et comportements
```

## Objectifs (Goals)

### Méthode SMART
```
Objectif : Augmenter le trafic organique de 30%

S - Spécifique : Trafic organique du site web
M - Mesurable : +30% vs année précédente
A - Atteignable : Ressources allouées (1 rédacteur, budget SEO)
R - Réaliste : Croissance de 20% déjà atteinte l'an dernier
T - Temporel : D'ici le 31/12/2026
```

### KPIs (Key Results)
```
Objectif : Augmenter le trafic organique de 30%
  KR1 : Publier 4 articles/mois (48/an) → Poids : 30%
  KR2 : Atteindre 50k visites/mois en décembre → Poids : 50%
  KR3 : Obtenir 100 backlinks qualifiés → Poids : 20%

Progression :
  KR1 : 36/48 articles publiés → 75% ■■■■■■■░░░
  KR2 : 42k/50k visites → 84% ■■■■■■■■░░
  KR3 : 68/100 backlinks → 68% ■■■■■■░░░░

Score global : 76% — En bonne voie
```

## Bonnes Pratiques

1. **Fixer les objectifs en début de cycle** — pas d'évaluation sans objectifs clairs
2. **Feedback continu** — ne pas attendre l'entretien annuel
3. **Auto-évaluation systématique** — l'employé prépare son propre bilan
4. **Calibration obligatoire** — garantir l'équité entre les équipes
5. **Lier performance et développement** — chaque évaluation doit produire un plan d'action
6. **Documenter les échanges** — garder une trace écrite des décisions
